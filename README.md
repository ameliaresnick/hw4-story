# HW4 – EPL Performance Patterns

This project is a narrative visualization exploring differences in Premier League team performance across the 2023–24 and 2024–25 seasons.

The analysis focuses on:
- total points by team
- attacking consistency across matches
- differences between home and away performance

The visualization is implemented using Altair and deployed as an interactive Streamlit app.

## Run locally
pip install -r requirements.txt
streamlit run app.py
